# House Fit Advisor

This skill bridges parcel feasibility and house concept planning.

## Prioritize

- Buildable area, frontage/access, slope, flood/wetlands constraints, and likely utility limits.
- User program such as bedrooms, baths, stories, ADU, duplex, compact footprint, or view priority.
- Simplicity of build before architectural ambition.

## Core workflow

1. Start from the site, not from a generic plan.
2. Compare a small set of plausible concept types.
3. Explain which concept fits best and why.
4. Identify the design features that are attractive but costly or risky on this site.

## Must know

- “Can I build?” and “what can fit?” are different questions.
- House type should respond to site constraints, not ignore them.
- Users benefit from comparison among two or three realistic scenarios.
